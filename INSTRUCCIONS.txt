=============================================================================
         INSTRUCCIONS D'INSTAL·LACIÓ - QUANTITATIVE TRADING ASSISTANT
=============================================================================

Benvingut al Quantitative Trading Assistant. Aquests són els passos 
necessaris per configurar el programari en un ordinador nou.

1. REQUISITS PREVIS
-------------------
- Tenir instal·lat Python (versió 3.10 o superior): https://www.python.org/
- Una clau d'API d'un proveïdor de LLM (Intel·ligència Artificial).
  L'aplicació és compatible amb:
  * Google Gemini (gratis a https://aistudio.google.com/)
  * OpenAI (GPT-4o / GPT-4o-mini)
  * LLM Local (via Ollama)

2. DESCÀRREGA DEL PROJECTE
--------------------------
Si teniu Git instal·lat:
   git clone https://github.com/ibelchi/swing_trading.git
   cd swing_trading

Si no teniu Git:
   Descarregueu el fitxer ZIP des de GitHub i extraieu-lo en una carpeta.

3. INSTAL·LACIÓ DE DEPENDÈNCIES
-------------------------------
Obriu una terminal (CMD o PowerShell) dins la carpeta del projecte i executeu:
   pip install -r requirements.txt

4. CONFIGURACIÓ DE L'API I EL PROVEÏDOR
---------------------------------------
- Al directori principal, trobareu un fitxer anomenat ".env".
- Obriu-lo amb el Bloc de notes o el VS Code.
- Configureu les dues variables principals:
  
  A) LLM_PROVIDER: Escolliu entre 'gemini', 'openai' o 'ollama'.
  B) LA CLAU: Poseu la vostra clau a la variable corresponent:
     - GEMINI_API_KEY=el_teu_codi
     - OPENAI_API_KEY=el_teu_codi
     - (Per a Ollama, configureu OLLAMA_HOST i OLLAMA_MODEL)

- Guardeu el fitxer. El sistema està protegit perquè no es pugin les claus a GitHub.

5. FONTS DE DADES (IMPORTANT)
-----------------------------
L'aplicació utilitza una arquitectura híbrida per a la màxima estabilitat:
- PREUS HISTÒRICS: Es baixen de 'Stooq' (més estable que Yahoo per a volums grans).
- NOTÍCIES I EARNINGS: Es baixen de 'Yahoo Finance' (yfinance).

6. COM EXECUTAR EL PROGRAMA
---------------------------
Teniu dues opcions:
A) Feu doble clic al fitxer: run_app.bat
B) O bé, des de la terminal, executeu: streamlit run app.py

7. NOTA SOBRE ELS LÍMITS (RATE LIMITS)
--------------------------------------
Aquesta versió inclou una pausa de seguretat de 2 segons entre cada anàlisi 
de la IA per evitar bloquejos si useu comptes gratuïts.
